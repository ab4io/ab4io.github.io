Lately, I have been watching a lot of cool projects from these amazing guys, one of which was even published on Y Combinator's website 'Hacker News'.  Let's take a look at a few of them.

## 1. **[sql-studio](https://github.com/frectonz/sql-studio/releases/tag/0.1.10 "sql-studio")** by [t.me/thefrectonz](https://t.me/thefrectonz "t.me/thefrectonz")

![](</sql studio.jpg>)

A single binary, single command SQL database explorer supporting SQLite, libSQL, PostgreSQL, MySQL, and DuckDB. It supports the following features:

* Overview page with common metadata
* Tables page with table metadata and disk usage
* Infinite scroll rows view
* Custom query page for more access
